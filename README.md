## 実装した機能・デザインの詳細
実装した機能は
* 直前の単語を、表示できるようにする
* 任意の単語を、入力できるようにする
* 直前の単語の末尾と、入力した単語の先頭を比較して、同じ場合だけ単語を更新する。違う場合は、エラーを表示する
* 末尾が「ん」で終わる単語が入力されたら、ゲームを終了する
* 過去に使用した単語が入力されたら、ゲームを終了する
* ゲーム中や終了後に、最初からやり直せるリセット機能をつける
  にくわえて
* ひらがな以外は入力できないようにする
* しりとりの単語の履歴を表示できるようにする
  の二つの機能を追加しました。

### 機能・デザインの詳細
* 直前の単語を、表示できるようにする機能
変数previousWordに直前で入力した単語を上書きしていき、各時点での変数previousWord内の単語を表示するようにしました。previousWord = "しりとり"とすることで最初の単語をしりとりにするようにしました。

* 任意の単語を、入力できるようにする機能
入力フォームに入力された値を JavaScriptのvalueで取得して、送信ボタンが押されたタイミングでサーバーへPOST /shiritoriでデータを送信することで単語を、入力できるようにしました。

* 直前の単語の末尾と、入力した単語の先頭を比較して、同じ場合だけ単語を更新し、違う場合は、エラーメッセージを表示する機能
server.jsで、previousWord.slice(-1) !== nextWord.slice(0, 1)を使って前の単語の最後と次の単語の最初を比較して、一致しない場合はエラーメッセージを表示するようにしました。

* 末尾が「ん」で終わる単語が入力されたら、ゲームを終了する機能
nextWord.slice(-1) === "ん"という条件式を使って、末尾の文字をチェックして、んを検出すると、エラーメッセージを表示して、ゲームを終了するようにしました。

* 過去に使用した単語が入力されたら、ゲームを終了する機能
送信した単語を格納する配列wordHistoriesをつくり、新しく入力された単語がすでに含まれているかをwordHistories.includes(nextWord)で判定して判断するようにしました。

* ゲーム中や終了後に、最初からやり直せるリセット機能
POST /resetを追加して、このリクエストを受け取ると単語の履歴やデータをリセットするようにしました。リセット」ボタンを追加して、location.reload()でサイトを更新できるようにしました。

* ひらがな以外は入力できないようにする機能
  ひらがな以外は入力できないようにするためにserver.jsにconst hiraganaRegex = /^[ぁ-ん]+$/;をついかすることで文字列の最後の文字まであ～ん中の文字しか使えないようにしました。また、ひらがな以外を使うとエラーメッセージを表示するようにしました。

* しりとりの単語の履歴を表示できるようにする機能
GET /historyを作成し、配列に保存されている単語の履歴を表示できるようにしました。送信が成功した瞬間にJavaScriptを使ってdocument.createElement("li")で新しいリスト要素を作り、サイトを更新しなくてもでリアルタイムに履歴を追加できるようにしました。
